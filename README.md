👉 [Chat](chat-ochre-alpha-83.vercel.app)

## 🛠️ Requirements

- [Node](https://nodejs.org/)
- [Git](https://git-scm.com/)

---

## 💻 Local Setup

Clone the repository
```bash
    git clone https://github.com/AndresOrozcoDev/chat.git
    cd chat
```

Install dependencies
```bash
    npm install
```

Run the development server
```bash
    npm run dev
```

## 🛠️ Requerimientos

- [Node](https://nodejs.org/)
- [Git](https://git-scm.com/)

## 💻 Ejecucion local

Clonar el repositorio
```bash
    git clone https://github.com/AndresOrozcoDev/chat.git
    cd chat
```

Instalar las dependencias
```bash
    npm install
```

Ejecutar el servidor
```bash
    npm run dev
```
